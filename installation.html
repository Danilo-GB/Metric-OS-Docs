Installation

Installation Process - Frontend - Developers

The following process will be used to install Metric OS frontend code.

  • Clone or download the repository: Github Link
  • Install Yarn from https://yarnpkg.com
  • Extract downloaded project files : Extract the Metric OS project to your suitable directory or folder.
  • Access console : Open node.js console.
  • In console, navigate to the location where main folder is extracted.
  • Navigate to the Source folder inside main folder.
  • Run yarn install or npm install command to install package dependencies.
  • Run yarn run dev or npm run dev to start compiling assets or yarn run build or npm run build for production use.
  • The frontend aplication needs the backend project in order to retrieve data.

Installation Process - Frontend - Final users

The following process will be used to install Metric OS frontend desktop application.

  • Download the zip file: Github Link
  • Extract downloaded project files: Extract the Metric OS folder to your suitable directory or folder.
  • Run the install file: metric-os Setup 1.0.0.exe
  • The frontend aplication needs the backend project in order to retrieve data.

Installation Process - Backend - All

The following process will be used to install Metric OS backend project.

  • Clone or download the repository: Github Link
  • Extract downloaded project files: Extract the Metric OS folder to your suitable directory or folder.
  • Install Composer from https://getcomposer.org/download/
  • In a console, navigate to the location where main folder is extracted.
  • Run: composer install
  • Recommendation: Use laragon as your development environment in order to execute apache and mysql servers.
  • We are ready!
